Linguagem C

Variáveis

Variáveis são como "caixas" onde você pode guardar informações
que seu programa precisa usar. Cada variável tem um nome e pode armazenar diferentes tipos de dados, como números ou textos.

Tipos de variáveis

  • int (inteiro)
  • float (números com ponto flutuante: 0.3, 1.45, etc)
  • char (caracter)

Boas práticas

Seguir boas práticas na declaração de variáveis em C (ou em qualquer linguagem de programação) é fundamental para garantir que o código seja legível, manutenível e livre de erros. Aqui estão algumas boas práticas a serem consideradas:

  • Use nomes de variáveis que descrevam claramente seu propósito. Evite abreviações excessivas.
  • Escolha o tipo de variável que melhor se adapta aos dados que você precisa armazenar. Use int para números inteiros, float ou double para números com ponto flutuante, char para caracteres, etc.
  • Sempre inicialize suas variáveis antes de usá-las. Variáveis não inicializadas podem conter valores indesejados.
  • Adicione comentários explicativos, especialmente se a variável não for autoexplicativa.

Entrada e saída de dados


Vale a pena saber que...

Quando você utiliza scanf para ler dados do teclado, o que realmente acontece é que o programa lê os dados de um "buffer" de entrada. Isso significa que, quando você digita algo e pressiona "Enter", o que você digitou é armazenado temporariamente na memória até que scanf seja chamado para processá-lo.
A função scanf ignora automaticamente os espaços em branco (como espaços, tabulações e quebras de linha) antes de ler os dados. Isso significa que, ao ler um número, você pode inserir espaços antes ou depois do número, e scanf ainda irá capturá-lo corretamente.

Atenção a explicação

  • printf é usado para mostrar uma mensagem ao usuário, solicitando que ele insira um valor.
  • scanf é usado para ler a entrada do usuário. Você deve passar o endereço da variável onde deseja armazenar o valor (usando o operador &).
  • Após a entrada, printf é usado novamente para mostrar os valores que o usuário digitou.

Condicionais

As condicionais em C permitem que você execute diferentes blocos de código com base em condições específicas. As principais estruturas condicionais em C são if, else if, e else. Além disso, o switch é outra forma de implementar a lógica condicional. Vamos explorar cada uma delas.

Estrutura "if"
A estrutura básica do if é usada para testar uma condição. Se a condição for verdadeira, o bloco de código dentro do if será executado.
Estrutura "else"
Você pode adicionar um bloco else que será executado se a condição do if for falsa.
Estrutura "else if"
Para testar múltiplas condições, você pode usar else if.
