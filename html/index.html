O que é uma linguagem de programação?

Um breve resumo sobre sobre lógica

Lógica de programação é o jeito de pensar para resolver problemas usando um passo a passo que o computador entende. É como criar uma receita de bolo, onde você dá instruções claras para que o computador siga e faça algo. Ela envolve:

  • Sequência: Fazer uma coisa depois da outra
  • Decisão: Escolher o que fazer baseado em condições (exemplo: se estiver quente, ligue o ventilador)
  • Repetição: Fazer algo várias vezes até que uma condição seja atendida (exemplo: repetir uma tarefa até acabar)
  • Operações Lógicas: Comparar coisas para tomar decisões (exemplo: "se isso for verdade, faça aquilo")
